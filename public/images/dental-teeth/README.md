# Imagens dos Dentes

Esta pasta contém as imagens dos dentes para o odontograma.

## Formato das Imagens

- **Formato**: PNG (recomendado) ou SVG
- **Nome do arquivo**: Use o número do dente seguido da extensão
  - Exemplo: `11.png`, `12.png`, `13.png`, etc.

## Numeração dos Dentes

### Dentição Permanente:
- **Superior Direita (Quadrante 1)**: 18, 17, 16, 15, 14, 13, 12, 11
- **Superior Esquerda (Quadrante 2)**: 21, 22, 23, 24, 25, 26, 27, 28
- **Inferior Esquerda (Quadrante 3)**: 31, 32, 33, 34, 35, 36, 37, 38
- **Inferior Direita (Quadrante 4)**: 41, 42, 43, 44, 45, 46, 47, 48

### Dentição Decídua:
- **Superior Direita (Quadrante 5)**: 55, 54, 53, 52, 51
- **Superior Esquerda (Quadrante 6)**: 61, 62, 63, 64, 65
- **Inferior Esquerda (Quadrante 7)**: 71, 72, 73, 74, 75
- **Inferior Direita (Quadrante 8)**: 81, 82, 83, 84, 85

## Características das Imagens

- **Vista**: Lateral (perfil do dente)
- **Orientação**: 
  - Dentes superiores: raízes para cima
  - Dentes inferiores: raízes para baixo
- **Fundo**: Transparente (PNG com alpha) ou branco
- **Tamanho recomendado**: 200x400px ou proporção similar

## Como Adicionar

1. Adicione as imagens nesta pasta com o nome do número do dente
2. Exemplo: Para o dente 11, adicione `11.png`
3. O sistema detectará automaticamente e usará a imagem
4. Se a imagem não existir, será usado o SVG padrão
